# Origen Media Proposals

Live proposal app for Origen Media. Built with Vite, React, Tailwind, and Supabase.

## Setup

```bash
npm install
```

Create a `.env.local` file with your Supabase credentials:

```
VITE_SUPABASE_URL=https://gkxwmkprgixbiwkpaxud.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key_here
```

```bash
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Connect repo in Vercel
3. Add environment variables in Vercel dashboard:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
4. Deploy
5. Add custom domain: proposals.origen.media

## How it works

Each proposal lives at `proposals.origen.media/[slug]`. The slug is generated by the n8n workflow after processing a clarity call transcript. The React app reads from Supabase by slug and renders the full cinematic proposal page.
